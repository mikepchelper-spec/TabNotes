Optional Drive backup

When enabled, backups go directly to your private Google Drive app data folder.

For OAuth review

TabNotes requests Google access only for the optional Google Drive backup feature. It uses the limited drive.appdata scope so backup files stay in the user's private app data folder and are not visible to the developer.
